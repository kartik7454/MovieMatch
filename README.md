# 🎬 MovieMatch

**MovieMatch** is a web-based application designed to help film enthusiasts discover, track, and manage their favorite movies. Whether you're looking for what's currently playing in theaters or building a personal movie library, MovieMatch provides a centralized hub for your cinematic journey.

---

## 🍿 Key Features

### 🔍 Discover Movies
- Explore **Now Playing** and **Trending** sections
- Stay up to date with the latest releases

### 📂 Personal Lists
Organize your movie experience with:
- **Watchlist** – Movies you plan to watch
- **Watched** – Movies you've already seen
- **Liked** – Your all-time favorites

### 🔎 Search
- Quickly find movies using the built-in search functionality

### 📄 Detailed Information
- View individual movie pages with in-depth details

### 👤 Personalized Profiles
- Create an account and access your lists from any device

---

## 🛠️ How It Works

### 🔐 User Accounts
- Secure authentication (sign in / sign out)
- Personalized movie tracking

### 🌐 Real-Time Data
- Fetches up-to-date movie data (titles, posters, release dates)

### 🗄️ Database Integration
- Stores user data securely:
  - Watchlist
  - Watched
  - Liked

---

## 🚀 Getting Started

1. **Sign In**  
   Create or log into your account

2. **Browse**  
   Explore trending and currently playing movies

3. **Track**  
   Add movies to:
   - Watchlist
   - Watched
   - Liked

4. **Manage**  
   View and edit your collections anytime
